Active la transcription et le résumé IA.

Colle ta clé API Groq. Elle reste stockée localement sur ton appareil. Quand tu transcris ou résumes, l’app appelle directement Groq depuis ton navigateur.

Garde cette app privée si tu utilises une clé dans le navigateur. Pour une app publique, il faudra passer par un backend sécurisé.

Installer l’app

Ajoute CoursMemo AI à ton écran d’accueil.

Si le bouton d’installation automatique n’apparaît pas, utilise le menu de ton navigateur puis choisis “Ajouter à l’écran d’accueil”.

  • Android : menu du navigateur → Ajouter à l’écran d’accueil
  • iPhone : partager → Sur l’écran d’accueil
